BotForge Widget Documentation

The BotForge Widget is a universal chatbot widget that works with all modern frameworks and vanilla JavaScript. It provides a seamless way to integrate AI-powered chat functionality into any website.

Installation

Method 1: HTML Embed (Recommended)

<!-- Add this before closing </body> tag --> <div id="botforge-widget"></div> <script> window.BotForgeConfig = { chatbotId: "your-chatbot-id", apiUrl: "https://botforge.site", position: "bottom-right", theme: "light", primaryColor: "#3B82F6" }; </script> <script src="https://botforge.site/widget/botforge-widget.js" async></script>

Method 2: NPM Package

npm install @botforge/widget import BotForgeWidget from '@botforge/widget'; const widget = new BotForgeWidget({ chatbotId: 'your-chatbot-id', apiUrl: 'https://botforge.site' }); widget.mount();

Configuration Options

Option Type Default Description
chatbotId string required Your unique chatbot identifier
apiUrl string required BotForge API endpoint
position string "bottom-right" Widget position: "bottom-right", "bottom-left", "top-right", "top-left"
theme string "light" Theme: "light", "dark", "auto"
primaryColor string "#3B82F6" Primary color for the widget
size string "medium" Widget size: "small", "medium", "large"
greeting string "Hi! How can I help you today?" Initial greeting message
placeholder string "Type your message..." Input placeholder text
showBranding boolean true Show "Powered by BotForge" branding
borderRadius number 12 Border radius in pixels
shadow boolean true Enable drop shadow
animation string "bounce" Animation: "none", "bounce", "pulse", "slide"
zIndex number 9999 CSS z-index value

API Reference

widget.mount()

Mounts the widget to the DOM. Called automatically if window.BotForgeConfig is available.

widget.open()

Opens the chat interface.

widget.close()

Closes the chat interface.

widget.toggle()

Toggles the chat interface open/closed state.

widget.destroy()

Removes the widget from the DOM and cleans up event listeners.

widget.updateConfig(newConfig)

Updates the widget configuration dynamically.

widget.updateConfig({ theme: 'dark', primaryColor: '#10B981' });
widget.getState()

Returns the current widget state including open status, messages, and configuration.

Event System

widget.on(event, callback)

Listen for widget events.

widget.on('open', () => console.log('Chat opened')); widget.on('close', () => console.log('Chat closed')); widget.on('message', (data) => console.log('Message:', data)); widget.on('mounted', (data) => console.log('Widget mounted:', data)); widget.on('destroyed', () => console.log('Widget destroyed'));

Framework Integration

React

import React, { useEffect } from 'react'; const BotForgeWidget = ({ chatbotId, config = {} }) => { useEffect(() => { window.BotForgeConfig = { chatbotId, apiUrl: 'https://botforge.site', ...config }; const script = document.createElement('script'); script.src = 'https://botforge.site/widget/botforge-widget.js'; script.async = true; document.body.appendChild(script); return () => { const existingScript = document.querySelector('script[src*="botforge-widget.js"]'); if (existingScript) existingScript.remove(); const widget = document.getElementById('botforge-widget'); if (widget) widget.remove(); }; }, [chatbotId, config]); return <div id="botforge-widget" />; };

Vue.js

<template> <div id="botforge-widget"></div> </template> <script> export default { props: ['chatbotId', 'config'], mounted() { window.BotForgeConfig = { chatbotId: this.chatbotId, apiUrl: 'https://botforge.site', ...this.config }; const script = document.createElement('script'); script.src = 'https://botforge.site/widget/botforge-widget.js'; script.async = true; document.body.appendChild(script); }, beforeUnmount() { const script = document.querySelector('script[src*="botforge-widget.js"]'); if (script) script.remove(); } }; </script>

Examples

Basic Integration

<!DOCTYPE html> <html> <head> <title>My Website</title> </head> <body> <h1>Welcome to my website</h1> <!-- BotForge Widget --> <div id="botforge-widget"></div> <script> window.BotForgeConfig = { chatbotId: "your-chatbot-id", apiUrl: "https://botforge.site" }; </script> <script src="https://botforge.site/widget/botforge-widget.js" async></script> </body> </html>

Custom Styling

window.BotForgeConfig = { chatbotId: "your-chatbot-id", apiUrl: "https://botforge.site", position: "bottom-left", theme: "dark", primaryColor: "#10B981", size: "large", greeting: "Welcome! How can we help you today?", placeholder: "Ask us anything...", showBranding: false, borderRadius: 20, animation: "pulse" };

Programmatic Control

// Initialize widget const widget = new BotForgeWidget({ chatbotId: 'your-chatbot-id', apiUrl: 'https://botforge.site' }); widget.mount(); // Control programmatically document.getElementById('open-chat').addEventListener('click', () => { widget.open(); }); document.getElementById('close-chat').addEventListener('click', () => { widget.close(); }); // Listen for events widget.on('message', (data) => { console.log('New message:', data); // Send to analytics, etc. });
